# Palette Assignment (GBC Tips)

You have **8 background palettes** and **8 sprite palettes**, each with 4 colors.

Suggested mapping:
- **Background Palettes**
  - P0 Roads: light gray → dark gray
  - P1 Grass/Park: light green → dark green
  - P2 Buildings: beige → brown
  - P3 Water: light blue → dark blue

- **Sprite Palettes**
  - S0 Hole: black → bluish gray
  - S1 Cars: red set
  - S2 Cars: blue set
  - S3 UI digits/icons: yellow set

**Note**: You’ll assign palettes inside GB Studio's scene palette inspector.
Make sure your hole center uses your ground's lightest background color to fake transparency.
